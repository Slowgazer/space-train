# 太空列车物语 — 决策记录

## 决策记录

### 决策 #001 — 渲染方案

| 字段 | 内容 |
|------|------|
| 日期 | 2026-06-09 |
| 议题 | 采用哪种渲染方式 |
| 选项 | Canvas 2D / PixiJS / Three.js / Phaser |
| 选择 | **Three.js + 正交相机** |
| 原因 | 需要大量粒子效果（拖尾、激光、破碎、小游戏），WebGL 性能更好 |
| 未选 | Canvas 2D（粒子性能不足）、PixiJS（Three.js 粒子系统更成熟）、Phaser（框架重） |

### 决策 #002 — 操作方式

| 字段 | 内容 |
|------|------|

### 决策 #005 — 种子位置

| 字段 | 内容 |
|------|------|
| 日期 | 2026-06-09 |
| 议题 | 种子在场景中位置 |

### 决策 #006 — 列车结构

| 字段 | 内容 |
|------|------|
| 日期 | 2026-06-09 |
| 议题 | 列车结构和玩家移动范围 |
| 选择 | 从右到左：车头 → 升级车厢 → 种子车厢 → 普通车厢；玩家在 Z 轴上层漂浮移动，不与列车碰撞 |
| 未选 | 整块平台、车厢间跳跃 |
