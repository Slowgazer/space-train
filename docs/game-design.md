## 1. 游戏总览

### 1.1 概念

玩家扮演一列在太空中航行的列车上的成员，主要任务是运输一个称为「种子」的特殊生命体。在航行过程中，玩家需清除阻挡航线的陨石和怪物，保护列车和种子的安全，同时通过升级系统和种子互动不断增强实力。

### 1.2 核心循环

```
自动向右航行 → 遭遇陨石/怪物 → 清除 → 获得货币
→ 靠近升级车厢升级 → 种子产出生命活性 → 靠近种子对话
→ 遇到站点/事件 → 交易/对话 → 继续航行
```

### 1.3 游戏结束条件

- 种子 HP 归零 → 游戏结束
- 列车 HP 归零 → 游戏结束
- 玩家 HP 归零 → 可从存档点复活（待定）

## 2. 技术选型

### 2.1 已选方案

| 项目 | 选择 | 原因 |
|------|------|------|
| 渲染引擎 | Three.js + 正交相机 | 需要大量粒子效果（拖尾、激光、破碎等），WebGL 性能好 |
| 视角 | 俯视角 2D 平面 | 适合太空场景表现 |
| 语言 | TypeScript | 类型安全，后续维护友好 |
| 构建工具 | Vite | 快速 HMR，支持局域网访问 `--host 0.0.0.0` |
| 美术风格 | 风格化几何体 | 快速开发，预留贴图接口后期可替换 |
| 角色表现 | 圆形 + 颜文字 | 创建时可选颜色和颜文字表情 |
| 网络层 | 抽象接口（GameNetwork） | 本地 LocalNetwork 实现，后续切换 RemoteNetwork |

### 2.2 备选方案（未选）

| 方案 | 未选原因 |
|------|---------|
| Canvas 2D | 粒子效果性能不如 WebGL |
| PixiJS | Three.js 粒子系统更成熟，且后续 3D 扩展空间更大 |
| Phaser | 框架较重，自定义需求多时反而不灵活 |

## 3. 列车系统

### 3.1 结构

列车从右到左排列（车头在前）：

```
从右到左： [ 车头 ] — [ 升级车厢 ] — [ 种子车厢 ] — [ 普通车厢... ]
```

- **车头**（最右）：列车头部，多人模式下玩家可使用吸盘枪改变车头方向，其他车厢跟随
- **升级车厢**：玩家靠近后按交互键打开升级面板
- **种子车厢**：种子本体，体型大，占据一整节车厢
- **普通车厢**（种子左侧）：普通车厢，后续可扩展功能

### 3.2 移动

- **单人模式**：自动向右滚动，摄像头始终以玩家为中心
- **过场动画**：摄像头可脱离玩家进行特殊运镜
- **多人模式**（预留）：仍为自动滚动，但玩家可通过吸盘枪改变前进方向

### 3.3 碰撞

- 列车有碰撞体积，陨石撞击会减少列车 HP
- 玩家在 Z 轴上层，不与列车发生碰撞，可漂浮经过列车上方
- 设定解释：玩家可在太空中漂浮到列车上方移动

### 3.4 安全区

- 列车周围矩形范围为安全区
- 安全区内玩家不会进入倒计时
- 超出安全区开始倒计时，倒计时结束扣减玩家 HP
- 可通过消耗生命活性升级太空活动时间（倒计时长度）

## 4. 玩家系统

### 4.1 控制

| 操作 | 方式 |
|------|------|
| 移动 | WASD 键 |
| 瞄准 | 鼠标控制准星 |
| 射击 | 左键按住持续发射激光，方向 = 玩家 → 鼠标准星 |
| 交互 | F 键（靠近种子/升级车厢时） |

### 4.2 角色

- 表现方式：圆形 + 颜文字
- 创建角色时选择：
  - 圆形颜色
  - 颜文字表情（作为区分标识）

### 4.3 属性

| 属性 | 初始值 | 升级方式 |
|------|--------|---------|
| 生命值（HP） | 100 | 消耗生命活性升级 |
| 移动速度 | 基础值 | 消耗生命活性升级 |
| 太空活动时间 | 基础值（秒） | 消耗生命活性升级 |
| 激光射程 | 基础值 | 消耗货币升级 |
| 激光伤害 | 基础 DPS | 消耗货币升级（同时增加激光宽度） |

## 5. 清理系统

### 5.1 基础激光

- 按住左键持续照射
- 方向：从玩家位置指向鼠标准星方向
- 射程：固定值，可通过升级增加
- 伤害：持续 DPS，可通过升级增加
- 激光宽度随伤害等级提升而增加

### 5.2 吸盘枪（斥力枪）

> 名称待定，涉及引力与斥力机制

**操作流程**：
1. 按住左键：吸附附近一颗陨石（一次只能吸附一颗）
2. 陨石靠近到一定距离后自动捕获
3. 自动开始三段蓄力
4. 再次点击左键：以斥力发射被捕获的陨石
5. 蓄力越多 → 发射速度越快、射程越远、伤害越大
6. 命中目标造成全部伤害后陨石破碎

**多人模式**：
- 对队友造成固定 20 点伤害（玩家初始 HP 100）

**可升级项**：
| 升级项 | 说明 |
|-------|------|
| 蓄力段数 | 最多 5 段，5 段发射属性大幅提升 |
| 蓄力速度 | 减少每次蓄力所需时间 |
| 发射力度 | 提升速度、伤害、射程 |

### 5.3 安魂曲（狙击枪）

- 发射子弹，射程超远，伤害高
- 可获取远处特殊物品（特殊陨石打碎后获得特殊奖励）
- 陨石有碰撞体积，玩家无法直接穿过（除非特殊职业），需先清理路径

**限制**：
- 弹夹 6 发
- 换弹速度慢

**可升级项**：
| 升级项 | 说明 |
|-------|------|
| 弹夹容量 | 增加子弹数 |
| 换弹速度 | 减少换弹时间 |
| 连发 | 可连续射击 |

### 5.4 导弹

- 大范围、大伤害
- 冷却时间很长
- 用于应对无法解决的突发状况

**注意事项**：
- 如果玩家在爆炸范围内，也会对列车造成伤害（伤害值待定）
- 初始储备：1 颗

**可升级项**：
| 升级项 | 说明 |
|-------|------|
| 爆炸范围 | 大幅增加范围 |
| 储备数量 | 增加可携带导弹数 |

### 5.5 预留接口

后续可能添加更多武器类型，预留武器注册接口。

## 6. 敌人类型

| 类型 | 行为 | 说明 |
|------|------|------|
| 普通陨石 | 漂浮碰撞 | 各种形状（圆形、六边形、不规则三角形等），仅靠碰撞造成伤害 |
| 追踪怪 | 主动追踪种子 | 攻击种子，需优先清除 |
| 射击怪 | 远程攻击 | 攻击玩家或种子 |
| 护盾陨石 | 护盾+本体 | 需先击破护盾才能伤害本体 |
| 精英/Boss | 特殊行为 | 固定距离或特殊事件出现 |
| 其他 | — | 预留敌人类型扩展接口 |

**难度递增**：随航行距离增加，陨石密度提升，怪物生命值提升。

## 7. 种子系统

### 7.1 定位

- 种子是列车的一节车厢（第三节），体型大
- 靠近后按 F 键交互
- 种子 HP 较低，种子死亡 → 游戏结束

### 7.2 产出

- 持续产出「生命活性」资源
- 可查看当前产出效率和累计量

### 7.3 对话系统

- **触发方式**：靠近种子按 F 键
- **性格系统**：种子性格随机生成，不同性格有不同回应倾向
- **玩法**：选择对话分支，选对选项获得大量奖励
- **小游戏**：有时触发内置小游戏，胜利获得大量奖励（游戏暂停）
- **时间暂停**：与种子交互时游戏暂停，防止忙不过来
- **后续计划**：接入 AI 自由对话

### 7.4 种子升级

- 通过特殊物品升级种子
- 可升级方向：
  - 增加生命活性产出效率
  - 生成辅助清扫的助手
  - 其他拓展功能（预留）

## 8. 经济与升级系统

### 8.1 货币体系

| 货币类型 | 获取方式 | 用途 |
|---------|---------|------|
| 普通货币 | 清除陨石/怪物掉落 | 升级武器、维修列车、解锁特殊能力 |
| 生命活性 | 种子持续产出 | 升级玩家属性、延长太空活动时间、特殊事件消耗、特殊交易 |

### 8.2 升级项总览

**消耗普通货币的升级项**（在升级车厢交互）：

| 升级项 | 说明 |
|-------|------|
| 激光伤害 | 提升 DPS，同时增加激光宽度 |
| 激光射程 | 延长最大有效距离 |
| 列车 HP/护甲 | 列车更耐撞击 |
| 种子产出效率 | 提升生命活性产量 |
| 列车维修 | 修复列车受损部分 |
| 吸盘枪升级 | 蓄力段数/速度/力度 |
| 安魂曲升级 | 弹夹/换弹速度/连发 |
| 导弹升级 | 范围/储备数 |
| 特殊能力 | 解锁新能力（如闪现、护盾等） |

**消耗生命活性的升级项**（种子交互或升级车厢）：

| 升级项 | 说明 |
|-------|------|
| 玩家最大 HP | 增加生命值上限 |
| 玩家移动速度 | 提升 WASD 移动速度 |
| 太空活动时间 | 增加安全区外倒计时长度 |

### 8.3 升级入口

靠近第二节「升级车厢」，按 F 键打开升级面板（游戏暂停），面板中列出所有可升级项及其消耗。

## 9. 事件与站点系统

### 9.1 站点

- **出现方式**：单人模式每固定距离必定出现
- **表现形式**：大型星球
- **真空区域**：站点周围一定范围内无陨石/怪物
- **交互方式**：靠近后自动打开站点界面（游戏暂停）
- **站点内容**：
  - 交易（用货币/生命活性购买物品）
  - 随机事件（与站点相关）

### 9.2 航行事件

- **出现位置**：航线两侧的大型星球
- **单人模式**：不会出现在航线正前方
- **标记方式**：感叹号（!）提示
- **交互方式**：靠近后触发文字对话事件（游戏暂停）
- **事件形式**：文字对话选择分支，减少开发开销

### 9.3 星系生态系统（预留接口）

影响以下内容（后续实现）：
- 陨石种类
- 怪物类型
- 事件内容
- 站点交易货物

### 9.4 多人模式扩展

星轨航图 → 选择星系 → 进入该星系站点 → 在该星系航行 → 结束后返回航图

### 10.1 当前状态

## 11. HUD / UI

### 11.1 常驻显示

| 元素 | 说明 |
|------|------|
| 玩家 HP 条 | 当前生命值 |

| 界面 | 触发方式 | 行为 |

## 12. 技术架构

### 12.1 项目结构

```
src/
├── main.ts                # 入口
├── game/
│   ├── Game.ts            # 游戏主类
│   ├── GameLoop.ts        # 游戏循环
│   └── SceneManager.ts    # 场景管理
├── engine/
│   ├── Camera.ts          # 正交相机（跟随玩家）
│   ├── Input.ts           # WASD + 鼠标输入
│   ├── Physics.ts         # 碰撞检测
│   └── ParticleSystem.ts  # 粒子系统
├── entities/
│   ├── Player.ts          # 玩家（圆形+颜文字）
│   ├── Train.ts           # 列车（车头+升级+种子+普通车厢）
│   ├── Seed.ts            # 种子
│   ├── Asteroid.ts        # 陨石基类
│   └── Monster.ts         # 怪物基类
├── weapons/
│   ├── Laser.ts           # 基础激光
│   ├── SuctionGun.ts      # 吸盘枪
│   ├── Requiem.ts         # 安魂曲
│   ├── Missile.ts         # 导弹
│   └── WeaponBase.ts      # 武器基类（预留扩展）
├── systems/
│   ├── UpgradeSystem.ts   # 升级系统
│   ├── EconomySystem.ts   # 经济系统
│   ├── DialogueSystem.ts  # 对话系统
│   ├── EventSystem.ts     # 事件系统
│   └── GalaxySystem.ts    # 星系系统（预留接口）
├── ui/
│   ├── HUD.ts             # 常驻 HUD
│   ├── UpgradePanel.ts    # 升级面板
│   ├── DialoguePanel.ts   # 对话面板
│   └── StationPanel.ts    # 站点界面
├── network/
│   ├── types.ts           # 网络类型定义
│   ├── GameNetwork.ts     # 抽象接口
│   ├── LocalNetwork.ts    # 本地实现
│   ├── RemoteNetwork.ts   # 远程实现（预留）
│   └── index.ts           # 工厂函数
└── config/
    └── api.ts             # API 配置文件
```

### 12.2 核心接口约定

所有子系统通过接口通信，核心依赖网络抽象层 `GameNetwork`，游戏逻辑不耦合具体网络实现。
