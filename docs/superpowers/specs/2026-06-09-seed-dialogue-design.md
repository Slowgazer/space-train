# 种子对话系统设计文档

> 日期: 2026-06-09 | 状态: 待实现

---

## 概述

种子对话系统允许玩家靠近种子车厢时按 F 交互，触发文字对话。选择正确选项获得活性 + 随机 buff 奖励。

---

## 性格系统

种子开局时随机生成 4 种性格之一：

| 性格 | 风格 | 对话倾向 |
|------|------|---------|
| 开朗 (Cheerful) | 积极乐观 | 短句偏快，表情符号多 |
| 忧郁 (Melancholy) | 敏感多愁 | 太空安静感，需安抚 |
| 傲娇 (Tsundere) | 嘴硬心软 | 表面冷漠实际关心 |
| 沉默 (Silent) | 惜字如金 | 极简表达，全靠玩家猜 |

每种性格包含 2 段专属对话 + 1 段通用对话，每个对话 3 个选项（1 个正确）。

## 对话数据

```ts
interface DialogueEntry {
  text: string;        // 种子说的话
  options: {           // 3 个选项
    text: string;
    correct: boolean;
  }[];
  rewardMin: number;   // 正确奖励活性下限
  rewardMax: number;   // 正确奖励活性上限
  buffType?: BuffType; // 正确时触发的 buff
}

enum BuffType {
  SPEED,   // 疾风：移速 +50%，15s
  DAMAGE,  // 热力：激光伤害 +10，15s
}
```

选对：`rewardMin~rewardMax` 活性 + buff（15s）
选错：`10~20` 活性安慰奖

## 冷却

每次对话后冷却 30s（对话结束时开始计时），冷却期间靠近按 F 无反应或提示 "种子在休息…"。

## 参数汇总

| 参数 | 值 |
|------|-----|
| 性格数 | 4 |
| 每性格对话数 | 2 专属 + 1 通用 |
| 每对话选项数 | 3（1 正确） |
| 正确奖励活性 | 50~100 |
| 选错奖励活性 | 10~20 |
| 疾风buff | 移速 +50%，15s |
| 热力buff | 激光伤害 +10，15s |
| 对话冷却 | 30s |
