# 吸盘枪（Suction Gun）设计文档

> 日期: 2026-06-09 | 状态: 待实现

---

## 概述

吸盘枪是玩家的第二种武器（备选），通过 Q/E 键循环切换。左键使用当前武器。

核心玩法：吸引前方陨石 → 捕获 → 蓄力 → 发射为穿甲弹。

---

## 武器切换系统

### 操作

| 按键 | 行为 |
|------|------|
| Q | 切换到上一个武器 |
| E | 切换到下一个武器 |
| 左键 | 使用当前武器 |

### 架构

`Game.ts` 维护武器数组，每帧委托调用当前武器的 `update(dt)` 方法：

```ts
interface Weapon {
  update(dt: number, input: InputState, playerX: number, playerY: number, aimDir: {x:number,y:number}): void;
  onSwitchAway(): void;  // 切走时的清理
}
```

初始两个武器：`[Laser, SuctionGun]`，占位索引 0 和 1。

### 切换表现

- 切走吸盘枪时调用 `onSwitchAway()` → 释放已捕获陨石（不做发射）
- 屏幕角落或 HUD 显示当前武器图标/名称

## 吸盘枪状态机

```
     ┌──────────┐
     │   IDLE   │ ←── 初始/发射后
     └────┬─────┘
          │ 左键按下 + 前方50°有陨石
          ▼
     ┌──────────┐
     │ATTRACTING│ ←── 按住左键持续
     └────┬─────┘
          │ 陨石进入0.5单位
          ▼
     ┌──────────┐
     │ CAPTURED │ ←── 悬浮在准星方向，自动蓄力
     └────┬─────┘
          │ 再按左键
          ▼
     ┌──────────┐
     │ LAUNCHED │ ←── 陨石飞出，命中衰减
     └────┬─────┘
          │ 伤害预算归零 或 超出射程
          ▼
         IDLE
```

## 详细行为

### IDLE 状态
- 不做任何事
- 等待左键按下

### ATTRACTING 状态（按住左键进入）
- **扇形判定**：以玩家位置为顶点，朝向鼠标准星方向，左右各 25°（共 50°）
- **选择最近陨石**：扇形内距玩家最近的陨石成为目标
- **吸引力**：距离越近吸力越大，公式 `force = 3 / (distance + 0.5)`，使陨石向玩家加速
- **视觉**：
  - 目标陨石方向到玩家的粒子流
  - 扇形范围半透明弧形网格
- **释放**：松开左键 → 回到 IDLE，陨石恢复自由

### CAPTURED 状态（陨石进入 0.5 单位自动触发）
- 陨石被"捕获"，跟随玩家位置，悬浮在鼠标准星方向、距玩家 0.5 单位处
- 自动三段蓄力，无需额外操作
- 蓄力进度通过视觉反馈（颜色/光环变化）

| 段位 | 蓄力时间 | 伤害预算 | 飞行速度 |
|------|---------|---------|---------|
| 1段 | 从捕获算起 0.6s | 30 | 5 |
| 2段 | 从捕获算起 1.2s | 50 | 8 |
| 3段 | 从捕获算起 2.0s | 80 | 12 |

- 在 CAPTURED 状态下再次按左键 → 进入 LAUNCHED

### LAUNCHED 状态（按左键进入）
- 陨石从玩家位置沿鼠标准星方向飞出（初始速度取决于蓄力段位）
- **碰撞判定**：每帧检测与所有陨石的距离
- 命中时：
  1. 目标陨石受到 `min(弹药预算, 目标HP)` 伤害
  2. 弹药预算减去造成的伤害
  3. 发射陨石继续飞行（预算 > 0 时）
- 弹药预算 ≤ 0：
  1. 发射陨石爆炸（粒子效果 + 屏幕震动）
  2. 发射陨石从场景移除
  3. 回到 IDLE

## 视觉反馈

| 阶段 | 效果 |
|------|------|
| 吸引 | 粒子流从目标陨石流向玩家；半透明扇形弧形网格 |
| 捕获 | 陨石悬浮在准星前方；蓄力进度光环（绿→黄→红） |
| 发射 | 陨石拖尾粒子 |
| 命中 | 爆炸粒子 burst |
| 预算归零 | 大爆炸 particle burst + 屏幕震动 |

## 参数汇总

| 参数 | 值 | 说明 |
|------|-----|------|
| 扇形角 | 50° | 吸附检测范围 |
| 扇形检测距离 | 6 | 最大吸附距离 |
| 捕获距离 | 0.5 | 陨石距玩家多远算捕获 |
| 吸附力公式 | `3 / (dist + 0.5)` | 每帧施加的吸引力 |
| 蓄力 1 段 | 0.6s / 30 伤害 / 5 速 | |
| 蓄力 2 段 | 1.2s / 50 伤害 / 8 速 | |
| 蓄力 3 段 | 2.0s / 80 伤害 / 12 速 | |
| 最大射程 | 15 | 发射后超过此距离自动消失 |
