# 安魂曲（狙击枪）设计文档

> 版本: v1.0
> 日期: 2026-06-09
> 状态: 策划确认稿

---

## 1. 概述

安魂曲是第三把武器（武器索引 2，在 Laser[0] / SuctionGun[1] 之后，Missile[3] 之前），定位为高伤害远程狙击武器。Q/E 切换到安魂曲后，鼠标左键单发子弹，适合远程清除高威胁陨石。

## 2. 操作与视觉

### 2.1 瞄准与射击
- **瞄准方式**：鼠标准星指向，左键点击发射
- **射程指示**：鼠标准星在射程（20 单位）内变为红色，超出为白色
- **射击方式**：单发点射，每次点击发射一颗子弹
- **子弹目标**：如果鼠标在射程内，子弹飞向鼠标位置；如果鼠标在射程外，子弹朝鼠标方向飞 20 单位后消失

### 2.2 子弹表现
- **飞行方式**：子弹从玩家位置高速飞向鼠标指向位置
- **飞行时长**：0.3 秒到最大射程（20 单位）→ 速度 ≈ 66.67 单位/秒
- **子弹碰撞**：圆形碰撞体，命中路径上第一个陨石后消失
- **视觉**：
  - 子弹本身：一个小的亮色圆形 Mesh 或 Sprite
  - 拖尾效果：复用 ParticleBurst 粒子池，发射 Sprite 沿线拖尾
  - 命中效果：小范围爆裂粒子

### 2.3 换弹
- 弹夹 6 发，打空后自动进入换弹
- 换弹时间：5 秒
- 换弹期间不能射击
- 换弹进度在 HUD 上显示（进度条或文字倒计时）

### 2.4 HUD
- 当前弹量 / 弹夹容量 显示（如 "3/6"）
- 换弹时显示 "换弹中..." + 倒计时进度

## 3. 数值

| 属性 | 初始值 | 说明 |
|------|--------|------|
| 伤害 | 100 | 一击秒杀 radius=5 以下的陨石（陨石 HP = radius×20） |
| 射程 | 20 | 超远距离，约为激光（4）的 5 倍 |
| 弹夹容量 | 6 | 打空后自动换弹 |
| 换弹时间 | 5s | 换弹期间不可射击 |
| 子弹飞行速度 | ~66.67 单位/秒 | 0.3s 飞完 20 单位 |
| 子弹碰撞半径 | 0.2 | 与陨石碰撞检测 |

## 4. 可升级项（后续在升级面板添加）

| 升级项 | 效果 |
|-------|------|
| 弹夹容量 | 每级 +2 发 |
| 换弹速度 | 每级 -0.5s（最低 2s） |
| 连发 | 解锁后按住左键可连续射击（0.3s 间隔） |

## 5. 技术实现

### 5.1 文件结构
- `src/weapons/Requiem.ts` — 新 Weapon 接口实现
- `src/ui/RequiemHUD.ts`（或集成到 HUD）— 弹量/换弹显示

### 5.2 Requiem 类接口实现

```typescript
class Requiem implements Weapon {
  update(ctx: WeaponContext): void { /* ... */ }
  // 内部状态:
  // - 弹夹剩余
  // - 是否换弹中
  // - 换弹计时器
  // - 子弹飞行状态（位置、方向、速度）
}
```

### 5.3 子弹管理
- 单发子弹，同一时间最多一颗飞行弹
- 子弹飞行中用 requestAnimationFrame 更新位置
- 每帧检测与所有陨石的碰撞
- 命中或超出射程后消失

### 5.4 准星变色
- 在 Input 或 HUD 层计算鼠标世界坐标到玩家的距离
- 如果距离 ≤ 20，设置准星为红色，否则白色

### 5.5 粒子拖尾
- 子弹飞行时每帧调用 `ParticleBurst.emitAlongRay()` 或等效方法
- 浅色（白色/淡黄）粒子，快速淡出

## 6. 边界情况

- **换弹中点击左键**：无反应（不能射击）
- **弹夹还有余弹时换弹**：不支持主动换弹（只有自动换弹）
- **陨石在射程外**：子弹飞到射程边界自动消失
- **无陨石路径上**：子弹飞到射程边界消失

## 7. 不包含在本次实现的内容

- 特殊陨石（金色陨石，打碎额外奖励）
- 安魂曲专属升级项（弹夹/换弹/连发）
- 音效
