# 生命教育科普 Skill — 种子进化系统

## 核心原则

当涉及种子系统、植物进化、科普内容时，生成的内容必须符合真实植物学知识，同时将专业知识自然地融入游戏机制。

## 参考资料系统

本 skill 附带可扩展的知识库，AI 在生成植物相关内容时必须先查询参考文件：

### 知识库文件

| 文件 | 内容 | 格式 |
|---|---|---|
| `data/plants-database.csv` | 植物进化性状数据库（性状、案例、原理、游戏效果） | CSV |
| `references/` | 用户自行添加的参考资料（书籍摘录、文章等） | MD 或 TXT |

### AI 使用规则

生成任何植物/种子/进化相关内容前：
1. **优先从 `data/plants-database.csv` 中检索**匹配性状的科学描述
2. 如果 `references/` 目录中有用户添加的资料，**必须参考**其中的内容
3. 如果 CSV 和参考资料中都没有，才使用 AI 自身知识，但需标注 `（AI 生成，建议核实）`

### 进化描述格式

每次种子进化时，必须包含：

```
[进化性状名称]

🌱 进化效果：[游戏数值变化]

💡 真实世界科普：
- [真实植物案例：引用 CSV 数据]
- [进化原理的科学解释]
- [一句话为什么这种进化对生存有利]
```

## 如何添加参考资料（给其他人的说明）

你可以把自己的参考资料放在 `references/` 目录下，AI 会自动读取和使用。

### 方法一：添加书籍/文章摘录（最简单）

在 `references/` 中创建 Markdown 文件（`.md`），例如：

```
references/
├── 植物的生存智慧.md        ← 你的参考资料
├── 植物学基础-摘录.md       ← 你的参考资料
└── ...                      ← 任意添加
```

格式示例：

```markdown
# 来源：《植物知道生命的答案》- 第3章
## 关键内容：植物的触觉反应
当植物被触碰时，会释放...（此处粘贴你想让 AI 参考的内容）

## AI 使用说明
在生成卷须、触性相关游戏内容时参考本章内容。
```

### 方法二：添加到 CSV 数据库（更结构化）

直接编辑 `data/plants-database.csv`，追加新行：

```
性状名称,分类,真实植物案例,进化原理,游戏效果,详细科普,标签,来源
```
已有内容会自动保留。

### 方法三：引用整本书

新建 `references/书名.md`，按章节整理摘录。建议为每个重要概念写一段简短的 `## 标题`，方便 AI 检索。

## 对话引导规则

与种子对话 AI 交流时：
1. 玩家提到植物/种子相关话题 → 自然引入相关的科普知识
2. 科普要简短（1-2 句），不打断游戏节奏
3. 可以在对话结尾用"你知道吗？"引出有趣的冷知识
4. 如果玩家表现出兴趣，再展开更详细的讲解

## 科学严谨性检查

生成科普内容后自动检查：
- ✅ 正确：用适应/进化角度（"植物进化出...以应对..."）
- ❌ 避免：拟人化过度（"植物想要..."、"植物感觉..."）
- ❌ 避免：拉马克主义（"因为需要所以进化"）
- ✅ 正确：达尔文自然选择（"在干旱环境中，具有针状叶的个体存活率更高"）
- ✅ 数据准确：引用真实植物名称和特征
